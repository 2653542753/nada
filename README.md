## 安装 & 运行：

#### 通过 pip 安装(推荐)
`$ (sudo) pip install (--upgrade) nada`

OS X:  
`$ brew install mpg123`

Ubuntu:  
`$ sudo apt-get install mpg123`

#### 通过源码安装
`$ git clone https://github.com/ahonn/Nada.git`

`$ cd Nada`

`$ sudo python setup.py install`

#### 运行
`$ nada`

## 功能
- 最新期刊
- 分类期刊
- 搜索期刊
- 本地跨网站收藏音乐
- 下载音乐到本地

## 操作:

  键  | 功能
:----:|:-------
  q   | 退出到终端
  m   | 回到菜单
  l   | 前进
  h   | 后退
  k   | 上移
  j   | 下移
  u   | 上一页
  i   | 下一页
  [   | 上一首
  ]   | 下一首
space | 播放/暂停
  p   | 当前播放
  a   | 加入收藏
  r   | 移出收藏
  c   | 进入收藏
  d   | 下载到本地
