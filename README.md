### LuooMusic 

落网音乐命令行播放器。基于 python ，需要使用 `BeautifulSoup`, `requests` 模块。Just for fun～

#### 依赖模块安装
BeautifulSoup:

`$ sudo pip install beautifulsoup4`

requests:

`$ sudo pip install requests`

#### 终端下运行：
`$ python music.py`

#### 基本操作:

键   | 功能
----:|:---------
m    | 回到菜单
l    | 前进
h    | 后退
k    | 上移
j    | 下移
[    | 上一首
]    | 下一首
空格	 | 播放/暂停
